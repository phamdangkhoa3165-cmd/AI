=== BÁO CÁO CHI TIẾT THUẬT TOÁN MIN-CONFLICTS (CSP) ===
Trạng thái xuất phát: (2, 0)

Trạng thái gán ngẫu nhiên ban đầu: [0, 2, 2]

Bước 1: Trạng thái gán hiện tại [0, 2, 2] | Số xung đột = 2
  -> Chọn ngẫu nhiên biến (Trợ thủ) có thể gây xung đột: Biến 2
    + Thử giá trị 1: Xung đột = 2
    + Thử giá trị 2: Xung đột = 2
    + Thử giá trị 4: Xung đột = 1
  -> Chọn giá trị 4 cho biến 2 (Xung đột mới: 1)

Bước 2: Trạng thái gán hiện tại [0, 4, 2] | Số xung đột = 1
  -> Chọn ngẫu nhiên biến (Trợ thủ) có thể gây xung đột: Biến 3
    + Thử giá trị 0: Xung đột = 0
    + Thử giá trị 1: Xung đột = 1
    + Thử giá trị 2: Xung đột = 1
    + Thử giá trị 3: Xung đột = 1
    + Thử giá trị 4: Xung đột = 1
  -> Chọn giá trị 0 cho biến 3 (Xung đột mới: 0)

Bước 3: Trạng thái gán hiện tại [0, 4, 0] | Số xung đột = 0

=> [THÀNH CÔNG] Tìm thấy nghiệm Y = [0, 4, 0] sau 3 bước.
